# Akash Dalve — Portfolio Website

## 🚀 How to Host (Free Options)

### Option 1: GitHub Pages (Recommended - Free Forever)
1. Create a GitHub account at github.com
2. Create a new repository named: `akash-dalve.github.io`
3. Upload `index.html` to the repo
4. Go to Settings → Pages → Source: main branch
5. Your site goes live at: `https://akash-dalve.github.io`

### Option 2: Netlify Drop (Fastest - 30 seconds)
1. Go to https://app.netlify.com/drop
2. Drag & drop the entire `akash-portfolio` folder
3. Instant live URL! (e.g. random-name.netlify.app)
4. You can set a custom subdomain for free

### Option 3: Vercel (Professional)
1. Go to https://vercel.com
2. Sign up with GitHub
3. Import repo or drag folder
4. Live at yourusername.vercel.app
